# Composite Laboratorio

Implementación del patrón Composite en Java para simular un sistema de archivos.

## Ejecución

Ejecutar `Main.java`.
Se construye un árbol de directorios y se invoca `getSize()` desde la raíz (`MiPC`).

## Salida

Se muestra la estructura y el tamaño total:

```
+ MiPC (1500 kb)
   + Documentos (200 kb)
   + Multimedia (1300 kb)
```

## Resultado

1500 kb (suma total de archivos)
